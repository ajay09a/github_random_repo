# 🚀 GitHub Repository Finder

A simple NextJS app to search and explore random GitHub repositories by programming language.

## ✨ Features

* 🔍 Select a programming language
* 📦 Fetch random repositories
* ⭐ View stars, forks, and issues
* 🔄 Refresh to see new repos
